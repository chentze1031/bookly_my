## Design Notes

- All screenshots should use the actual app font (Helvetica Neue / NotoSansSC)
- Use the app icon colors consistently (dark #18160F as primary accent)
- Keep text large and readable - the top overlay text should be 36-42px bold
- Phone mockup should occupy about 60% of the vertical space
- Add subtle rounded corners to the phone mockup frame
- Backgrounds should have a gentle gradient, not flat color
- Each screenshot should be self-explanatory even without reading the description

## Tool Recommendation

Use Canva (free) or Figma to create these screenshots:
- Template search: "App Store screenshot template 1080x1920"
- Or use the app's actual screenshots as a base and add text overlay in Canva

Steps:
1. Run the app in an Android emulator or real device
2. Take clean screenshots of each screen mentioned above
3. Use Canva to add the phone frame, background color, and text overlay
4. Export as PNG, 1080x1920
5. Upload to Google Play Console
